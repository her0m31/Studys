\section{はじめに}

本アプリケーションはユーザが住所録の管理を行うためのものである。本アプリケーションを用いることで、ユーザは氏名、郵便番号、住所、電話番号、メールアドレスの5項目からなる住所録を管理することが可能となる。このアプリケーションでユーザが使用できる機能として、以下のものが挙げられる。
\begin{enumerate}
\item 新規住所録の追加
\item 既存住所録の削除
\item 既存住所録の参照
\item 任意の住所録の項目に対しての修正
\item 住所録の一覧表示
\end{enumerate}

また、本アプリケーションが自動で行う機能として、以下のものが挙げられる。
\begin{enumerate}
\item ファイルの読み込み
\item ファイルの保存
\item 履歴の一覧表示
\end{enumerate}

本アプリケーションを利用することによって、ユーザはファイルの管理を特殊なコマンドを入力することなく容易に行うことができる。

\section{アプリケーションについて}
\subsection{アプリケーションの起動}
\begin{verbatim}
本アプリケーションを起動するためには、まず端末エミュレータを起動する。「cd AB」と入力し、Enterを押す。すると次の行に進むので、「./Addressbook」と入力し、Enterを押す。これでアプリケーションが起動する。
正しく起動した場合は機能一覧が表示される。ユーザはその中の機能を番号を選択することによって選び、アプリケーションを実行する。
\end{verbatim}

\subsection{新規住所録の登録}

ここでは一つ目の機能である新規住所録の登録について説明する。新規住所録の登録は、ユーザが新しい住所録を登録するときに使う機能である。ユーザは説明にしたがって項目を入力することで住所録を登録できる。
\begin{enumerate}
\item 氏名の入力\\
登録したい住所録の最初の項目である氏名を入力する。姓名の間はアンダーバーを用いる。これがないと誤認識する。 (例: IWATE\_Taro)
\item 郵便番号の入力\\
登録したい住所録の郵便番号を入力する。(例: 0201231)
\item 住所の入力\\
登録したい住所録の住所を入力する。(例: Aoyama/Morioka/Iwate)
\item 電話番号の入力\\
登録したい住所録の電話番号を入力する。(例: 08012342222)
\item メールアドレスの入力\\
登録したい住所録のメールアドレスを入力する。(例: Iwate@iwate-uc.jp)
\end{enumerate}

\subsection{既存の住所録の検索}
ここでは二つ目の機能である既存の住所録の検索について説明する。ユーザは既に登録済みの住所録を任意の項目で検索することができる。
\begin{enumerate}
\item 検索する項目の選択\\
氏名、郵便番号、住所、電話番号、メールアドレスの5つの中から1つを選択し、数字で入力する。複数の項目を選択することはできない。
\item 検索キーの入力\\
検索するキーを入力する。
\end{enumerate}

\subsection{任意の住所録の項目の修正}
ここでは三つ目の機能である任意の住所録の項目への修正について説明する。ユーザは既に登録済みの任意の住所録の項目に対して修正をすることができる。
\begin{enumerate}
\item 検索する項目の選択\\
氏名、郵便番号、住所、電話番号、メールアドレスの5つの中から1つを選択し、数字で入力する。複数の項目を選択することはできない。
\item 検索キーの入力\\
検索するキーを入力する。
\item 一致した住所録一覧からの選択\\
検索キーに一致した住所録一覧が表示されるので、左側にある番号を入力して住所録を選択する。
\item 修正する項目の選択\\
修正する項目を番号で選択する。本アプリケーションでは一度にひとつの項目しか修正できない。
\item 修正後のデータの入力\\
修正する項目の修正後のデータを入力する。
\end{enumerate}


\subsection{既存住所録の削除}
ここでは四つ目の機能である既存住所録の削除について説明する。既存住所録の削除は、ユーザが既存の住所録を削除するときに使う機能である。ユーザは説明にしたがって項目を入力することで住所録を削除することができる。
\begin{enumerate}
\item 住所録の氏名の入力\\
ユーザは削除する住所録を検索するために、削除したい住所録の氏名を入力する。
\item 住所録の郵便番号の入力\\
ユーザは削除する住所録を検索するために、削除したい住所録の郵便番号を入力する。削除する住所録の検索はタイトル名と副タイトル名の完全一致で行う。
\item 削除の確認
削除対象の住所録が見つかったら、本当に削除するかどうかを選択する。ここで選択肢Noを選択すると削除をキャンセルすることができる。Yesを選択すると削除を実行する。
\end{enumerate}

\subsection{住所録の一覧表示}
ここでは五つ目の機能である住所録の一覧表示について説明する。住所録の一覧表示は、ユーザが住所録の一覧を見たいときに使う機能である。ユーザは説明にしたがって項目を入力することで住所録の一覧をみることができる。
\begin{enumerate}
\item 表示方法の選択\\
ユーザは一覧表示の方法を選択するため、表示にしたがって番号を入力する。
\item 項目の選択\\
ユーザは昇順または降順一覧表示を選択した場合は、どの項目について降順・昇順に表示するか選択する。表示にしたがって番号を入力する。
\end{enumerate}

\subsection{アプリケーションの終了と実行}
ここではアプリケーションの終了と実行について説明する。ユーザは本アプリケーションの機能を一度実行する毎にアプリケーションの実行を続けるか、終了するかを選択することができる。入力は選択肢の数字によるものとする。
