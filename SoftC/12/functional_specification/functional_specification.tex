\section{概要}
本アプリケーションはユーザの住所録の管理を行うためのものである。本アプリケーションを用いることで、氏名、郵便番号、住所、電話番号、メールアドレスの5項目からなる住所録を管理するこができる。アプリケーションの機能として、以下のものが挙げられる。\\
・作成した住所録の保存、読込\\
・新しい住所録の追加、不要な住所録の削除\\
・任意の住所録、項目に対しての修正\\
・任意の項目での検索、昇順並び替え、降順並び替え\\
・検索履歴表示\\

本アプリケーションの利便性として、住所録の管理が容易になることや「氏名」
「郵便番号」「住所」「電話番号」「メールアドレス」の情報をまとめて管理で
きることが挙げられる。さらに、検索履歴表示機能があることで
参照、検索時の手間が省けることがある。そしてユーザは端末エミュレータにUNIXコマンドを打ち込むことなく住所録の管理ができる。

\section{使用方法}
本アプリケーションはユーザが過去に保存した住所録を参照したいときや、新しく住所録情報を追加したいとき、住所録のデータを編集をするときに利用されると想定される。また、すべての機能はUNIXコマンドを入力することなく使用することができるので、ユーザは新たな住所の追加や削除なども容易に行うことができる。大まかな使用手順を次に示す。
\begin{enumerate}

\subsection{使用手順}

\item 機能一覧が表示される, 過去に検索した住所録があればこれも表示される(検索回数の多い最大5件)
\item ユーザは使用したい機能を選択する(機能名の隣に表示される番号で選択)
\begin{enumerate}


\item 新規住所録の登録
\begin{enumerate}
\item 登録したい住所録のデータを入力(氏名、郵便番号、住所、電話番号、メールアドレス)
\item 登録後の住所録一覧を表示
\end{enumerate}


\item 既存住所録データの参照\begin{enumerate}
\item 参照したい住所録を探すための項目を入力(氏名、郵便番号、住所、電話番号、メールアドレスの中から選択して入力)
\item 入力された項目に一致した住所録一覧を表示
\end{enumerate}

\item 既存住所録データの削除
\begin{enumerate}
\item 削除したい住所録を探すための項目を入力(氏名、郵便番号、住所、電話番号、メールアドレスの中から選択して入力)
\item 入力された項目に一致した住所録を表示
\item 本当に削除するかを選択
\item 指定した住所録削除後の住所録一覧を表示
\end{enumerate}

\item 任意の住所録、任意の項目に対しての修正
\begin{enumerate}
\item 修正したい住所録を探すための項目を入力(氏名、郵便番号、住所、電話番号、メールアドレスの中から選択して入力)
\item 入力された項目に一致した住所録一覧を表示
\item 修正する項目、修正後の住所録を入力
\item 修正後の住所録を表示
\end{enumerate}

\item 住所録の昇順・降順並び替え
\begin{enumerate}
\item 昇順または降順に並び替えたい項目を入力(氏名、郵便番号、住所、電話番号、メールアドレスの中から選択して入力)
\item 昇順と降順のどちらで並び替えるか入力
\item 並び替え後の住所録を表示
\end{enumerate}

\end{enumerate}

\end{enumerate}
